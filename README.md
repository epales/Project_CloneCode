## 6. 페이지 설명<br/>
**index.html**<br/>
> **사이트 메인 페이지**<br/>
> - 로그인 화면 이동, 내 상점, 번개톡 가능<br/>
> - 상품 판매하기 가능<br/>
> - 제목 검색, 카테고리 검색 가능<br/>
> - 최신등록 상품 무한스크롤 조회 가능<br/>

>**productDetail.html**<br/>

>**searchCategoryProduct.html**<br/>
> - 카테고리 검색 페이지<br/>
